## ทำนายยอดขายบน WooCommerce ด้วย Polynomial Regression และจัดกลุ่มลูกค้า ##
ส่วนเสริมสำหรับ WooCommerce ที่ช่วยให้ระบบสามารถ: 
- ทำนายยอดขายเดือนปัจจุบันและอีก 2 เดือนข้างหน้าด้วยโมเดล Regression แบบ Polynomial
- จัดกลุ่มลูกค้า Customer Clusters/Segmentation แบ่งกลุ่ม ลูกค้าทั้วไป ลูกค้าประจำ ลูกค้าพรีเมียม

## ระบบทำงานอย่างไร ##
###  ทำนายยอดขาย ###
ระบบจะดึงข้อมูลจากยอดขายแต่ละเดือนของ WooCommerce มาคำนวณ Polynomial Degree 2 ตามสูตรคือ y = ax^2 + bx + c 
ระบบจะทำการหาค่าสัมประสิทธิ์ a, b, c ออกมาสดๆ (ผ่านกฎของ Cramer's Rule) เพื่อให้ได้ "เส้นประแนวโน้ม" ที่โค้งดัดตามความสวิงของยอดขายในอดีต
### จัดกลุ่มลูกค้า ###
จัดกลุ่มลูกค้าโดย K-means กำหนดค่า K=3 เพื่อแบ่งกลุ่มลูกค้าออกเป็นสามกลุ่ม โดยจัดกลุ่มตามยอดสะสม ความถี่ในการซื้อ

## วิธีติดตั้งปลั้กอิน ##
ไปที่ <code>/wp-admin/plugin-install.php</code> และอัปโหลดไฟล์ zip ของปลั้กอิน จากนั้นสามารถ <b>Activate</b> แล้วเริ่มใช้งานปลั้กอินจากเมนูได้เลย !
